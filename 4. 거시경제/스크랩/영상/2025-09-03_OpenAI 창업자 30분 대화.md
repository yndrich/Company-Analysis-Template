## 1. 핵심 요약

- GPT5 기대 : 과학적, 기술적 질문에 훌륭한 답변, 프로그램을 만들어줌. 
- GPT5 영향 : GPT를 잘 사용하고, 말고가 이후의 미래를 바꿀 것 
- GPT에서 발견한 것 중 가장 인상깊었던 것 : 프로그래밍을 엄청 잘한다는 것. 
- GPT5 이후 미래 : 2027년 말까지는 AI로 의미있는 새로운 발견을 할 수 있을 것 
- 아직 부족한 부분 : 인지부분, 장기작업(시간이 많이 소요되는) 
- 초지능 이란? 인간의 역할이 필요없어질 때
	- 연구같은 부분을 전적으로 AI에 맡길정도 
- 지역, 문화 등의 차이를 모델이 어떻게 극복하는지?
	- 같은 모델을 쓰지만, 사용하는 사람에 대한 질문들의 기억을 갖고 있음. 각 사용자마다 맞춤화돼서 답변을 할 것 
- 5년안에 화이트칼라 직업들의 절반이 사라질 것, 2030년 다른 생각은 ?
	- 새로운 기술을 다시 배우고 싶어하지 않는 60대이상이 걱정
	- 1인 기업 시작, 10억 달러 이상의 가치를 세우는 회사를 차리기 좋은 환경이 될 것
	- 도구들을 사용하는 법을 배우고 아이디어만 떠올리면 됨 
- 기술적인 부분 : 어떻게 가장 강력한 지능을 만들고 있는지? 그리고 방해물은?
	- 컴퓨팅 파워 : 인프라 (반도체, 메모리, 네트워킹 장비, 에너지 등) 건설 우선시 되어야함. 
		- 에너지공급이 가장 큰 제약
	- 데이터 : 합성데이터 기대
	- 알고리즘 계산 : 수십, 수백개의 알고리즘 개선이 있을 것, 로컬에서 성능 좋은 모델을 돌릴수 있을만큼 알고리즘이 발달
- 어떻게 미래를 준비하냐? AI 도구를 계속해서 이용해보라. (내 삶에 어떻게 활용할지)
