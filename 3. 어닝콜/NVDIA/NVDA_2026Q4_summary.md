## 🗣️ 키워드 / 톤 

- 자주 언급된 단어
	- - **데이터센터 / AI 인프라 / AI 팩토리**  
	    → 사업 정체성(Identity) 자체를 “AI factory building blocks”로 정의
    
	- **Blackwell / Rubin / (전환, 램프, 샘플, 하반기 양산)**  
	    → 로드맵이 ‘마케팅’이 아니라 ‘실행 일정’으로 말해짐
    
	- **Inference(추론) / Tokens(토큰) / Tokenomics**  
	    → 고객의 수익화 모델을 NVIDIA 매출 성장 논리로 연결
    
	- **Performance per watt(와트당 성능) / Cost per token(토큰당 비용)**  
	    → 전력 제약 시대의 “경제성 KPI”를 선점
    
	- **CUDA / (TensorRT‑LLM)**  
	    → 소프트웨어 우위가 “추론 시대”에 더 중요하다는 논리
    
	- **NVLink / Spectrum‑X / InfiniBand / networking(네트워킹)**  
	    → GPU만으로는 AI 팩토리가 완성되지 않는다는 스택 강조
    
	- **Ecosystem(생태계) / Partnership(파트너십) / Investment(투자)**  
	    → 기술 우위 + 생태계 지배(시장 구조) 우위의 이중축
    
	- **Supply constraints(공급 제약) / purchase commitments(구매 커밋먼트)**  
	    → 리스크의 방향이 “수요 둔화”보다 “공급 확보/병목”으로 제시됨
	    
	- **Inflection point(변곡점)**  
	    → “에이전틱 AI가 최근 2~3개월 사이 변곡점 돌파”라는 시간표 제시
	    
- 경영진 톤
	- ### CFO(Colette) 톤: “수치·가이던스·리스크 관리”
		- 특징: **정량/가이던스 중심**, 단정적이되 “시점은 아직 이르다(too early)” 같은 표현으로 리스크를 관리
		- 대표 톤 포인트
		    - 중국 매출: “가이던스에 포함 안 함”으로 **불확실성 차단**
		    - Rubin 램프: “수요는 강하지만 램프 규모/시점은 아직 이르다”로 **과열 기대를 누름**
		    - 게이밍: “몇 분기 타이트, 연말 개선 가능성”처럼 **조건부/확률형 표현**
		→ 투자자 입장에선 “CFO가 낙관을 제어하는 브레이크” 역할을 하고 있습니다.

	- ### CEO(Jensen) 톤: “서사·확신·1원칙(First principles) + 강한 반복”
		- 특징: **반복(repetition)**, **은유(AI factory, 산업혁명)**, **단정적 확신(Compute=Revenue)**
		- 질문에 대한 답변 방식:    
		    - 숫자/구체치 요구에도, 종종 **프레임을 바꿔(재정의) 설득**합니다.
			    - 예: CapEx 지속가능성 질문 → “현금흐름은 성장할 것(확신)” + “토큰이 매출” 논리로 전환
			- 강점: 시장이 가장 궁금해 하는 “왜 이 사이클이 구조적이냐”에 대해  **고객의 수익화 논리**로 답을 구성합니다.
			- 약점(투자자 관점): 일부 답변은 **정량 구체성(예: Spectrum 런레이트)**이 부족하고,  “세계 최대” 같은 **강한 수사**가 들어가 검증 욕구를 자극할 수 있습니다.
